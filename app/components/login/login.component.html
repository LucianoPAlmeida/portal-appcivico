Ajuda

Algumas configurações iniciais são necessárias antes que o desenvolvedor possa utilizar os webservices da Nuvem Cívica em seu projeto de aplicativo. A primeira ação do desenvolvedor deve ser a criação de uma conta de usuário, que pode ser iniciada nessa página. Uma vez o cadastro bem-sucedido o usuário receberá seu código de usuário que poderá utilizar para cadastrar seus aplicativos e outras definições de ambiente para eles. Para mais informação acesse o GitHub do projeto Nuvem Cívica em https://github.com/AppCivicoPlataforma/AppCivico.
