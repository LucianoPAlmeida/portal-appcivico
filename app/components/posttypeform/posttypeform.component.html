Sobre Descrição de Conteúdos

Descrição detalhada sobre o tipo de conteúdo que será armazenado nesse tipo de postagem. É importante descrever de forma bem delhada o que cada campo dos conteúdos significa e qual a finalidade de armazenar cada um. Máximo 1000 cracteres.

Exemplo

Formato do Conteúdo

O tipo de postagem é compartilhamento de preço de remédio

Os conteúdos desse tipo de postagem serão em geral armazenados no formato:

{ "JSON": "{"latitude" : 31.3128903, "longitude": 12.8391083}", "valor": 10.32 }

Descrição do conteúdo

O conteúdo possui no campo valor o preço compartilhado pelo usuário e no campo JSON os valores de latitude e longitude que representam onde esse preco foi encontrado.
